# VoltGuard: محلل جودة الطاقة المدعوم بالذكاء الاصطناعي

تطبيق ويب حديث لتحليل وتوقع جودة الطاقة باستخدام تقنيات الذكاء الاصطناعي.

## المميزات

### واجهة المستخدم
- **شبكة إدخال (8x16):** إدخال 128 نقطة بيانات مع تأثير زجاجي
- **اللصق الذكي:** تحليل تلقائي للأرقام المفصولة بفواصل أو أسطر
- **تصور شكل الموجة:** مخطط خطي متحرك يعرض البيانات
- **بطاقة النتائج:** عرض فئة جودة الطاقة ونسبة الثقة

### التصميم
- **الألوان:** أزرق بحري (#0a2a4f) مع ذهب برتقالي (#f0a500)
- **الطباعة:** خطوط أحادية المسافة (JetBrains Mono و Roboto Mono)
- **التأثيرات:** توهج، شفافية، انتقالات سلسة

### الخادم
- **FastAPI:** خادم سريع وموثوق
- **CORS:** دعم الطلبات من أي مصدر
- **نقاط النهاية:**
  - `POST /predict` - التنبؤ بفئة جودة الطاقة
  - `POST /generate_test` - توليد عينة عشوائية
  - `GET /health` - فحص صحة الخادم

## التثبيت والتشغيل

### المتطلبات
- Node.js 22+
- Python 3.11+
- pnpm

### خطوات التثبيت

1. **تثبيت المكتبات:**
```bash
cd /home/ubuntu/voltguard
pnpm install
pip3 install fastapi uvicorn numpy
```

2. **بدء خادم FastAPI:**
```bash
python3 fastapi_server.py
```

3. **بدء خادم التطوير:**
```bash
pnpm dev
```

4. **الوصول إلى التطبيق:**
- الواجهة الأمامية: http://localhost:3001/
- وثائق API: http://127.0.0.1:8000/docs

## الاستخدام

### إدخال البيانات
1. اختر أحد الخيارات:
   - أدخل القيم يدويًا في الشبكة
   - استخدم "عينة عشوائية" لتوليد بيانات تجريبية
   - استخدم "اللصق الذكي" لنسخ 128 قيمة

2. اضغط على "تحليل شكل الموجة"

3. اعرض النتائج في بطاقة النتائج

## نقاط النهاية (API Endpoints)

### POST /predict
التنبؤ بفئة جودة الطاقة

**الطلب:**
```json
[0.5, 0.6, 0.7, ..., 0.8]  // 128 قيمة
```

**الاستجابة:**
```json
{
  "prediction": 1,
  "confidence": 0.85,
  "class_name": "جيد",
  "message": "تم التنبؤ بنجاح"
}
```

### POST /generate_test
توليد عينة عشوائية

**الاستجابة:**
```json
[0.5, 0.6, 0.7, ..., 0.8]  // 128 قيمة عشوائية
```

### GET /health
فحص صحة الخادم

**الاستجابة:**
```json
{
  "status": "healthy",
  "service": "VoltGuard Power Quality Analyzer"
}
```

## فئات جودة الطاقة

| الفئة | الوصف | اللون |
|------|-------|-------|
| 0 | جيد جداً | أخضر |
| 1 | جيد | أخضر فاتح |
| 2 | متوسط | أصفر |
| 3 | سيء | أحمر |
